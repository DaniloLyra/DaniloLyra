1. Encontre o domínio das funções:

a) \ $f(x) = 2x+1$\

Resp: $D = x \in \mathbb{R}$

b) \ $f(x) = \frac{x+3}{x+1}$\

Resp: \ $x+1 \ne 0$\ \ $x \ne -1$\ $D = \mathbb{R} - {-1}$

c) \ $f(x) = \sqrt{9 - x^2}$\

Resp: $9 - x^2 \ge 0$
$-x^2 \ge -9$
$x^2 \le 9$
$x \le \sqrt{9}$
$x \le \pm3$
$D = -3 \le x \le 3$
